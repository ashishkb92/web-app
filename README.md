## Steps to run this project

```sh
git clone https://github.com/ashishkb92/web-app.git
cd web-app
npm install
npm start
Open localhost:3000 in browser
```

## Framework/Technologies used

- React
- JSON-server
- CSS Modules
- Node (>16)
